🚀 Quick Setup

To test the account page with real backend integration:

  1. Install PostgreSQL (or use a cloud database)
  2. Create a database and update .env file
  3. Run database migrations: npm run db:schema
  4. Start the backend: npm start
  5. Open account.html in your browser

🗄️ Database Setup

If you have PostgreSQL installed locally:

# Create database
createdb universe_merch

# Or use a connection string like:
# DATABASE_URL="postgresql://username:password@localhost:5432/universe_merch"

Update your .env file:

DATABASE_URL="postgresql://your-connection-string"
JWT_SECRET="your-super-secret-jwt-key-here"
PORT=4000

🔗 API Endpoints

  • GET /api/auth/me - Get user profile
  • PATCH /api/auth/me - Update profile
  • GET /api/orders - List user orders
  • GET /api/addresses - List user addresses
  • POST /api/addresses - Create address
  • PATCH /api/addresses/:id - Update address
  • DELETE /api/addresses/:id - Delete address
